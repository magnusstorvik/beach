# Vikaneset photo naming

Use lowercase names, hyphens between words, and a two-digit sequence number. Keep the
original full-resolution files elsewhere and export web images as `.jpg`.

## Carousel folders and filenames

Place future carousel images in `assets/images/carousels/` using these names:

- `stay-sjohus-01.jpg`, `stay-sjohus-02.jpg`, `stay-sjohus-03.jpg`
- `stay-leilighet-01.jpg`, `stay-leilighet-02.jpg`, `stay-leilighet-03.jpg`
- `restaurant-klippfiskbrygga-01.jpg`, `restaurant-klippfiskbrygga-02.jpg`, `restaurant-klippfiskbrygga-03.jpg`
- `sauna-strand-01.jpg`, `sauna-strand-02.jpg`, `sauna-strand-03.jpg`
- `havn-01.jpg`, `havn-02.jpg`, `havn-03.jpg`
- `amfi-01.jpg`, `amfi-02.jpg`, `amfi-03.jpg`
- `galleri-01.jpg`, `galleri-02.jpg`, and so on

The first image should be the strongest overview. Use the second for an interior or
closer view and the third for a distinctive detail. Add higher sequence numbers when
a carousel needs more than three images; never reuse a number for a replacement.

Recommended web export: JPEG, sRGB, approximately 2000 px on the long edge, with a
target size below 500 KB where quality permits.
